# Plot 2D Fate Space Maturation

Generates a 2D scatter plot of lineage divergence, colored continuously
by Tocky Time (Blue -\> Red).

## Usage

``` r
plotTockyFate2D(object, ...)

# S4 method for class 'mCanonicalTockyObj'
plotTockyFate2D(object)
```

## Arguments

- object:

  An mCanonicalTockyObj with fate scores calculated.

- ...:

  Additional arguments passed to methods.
